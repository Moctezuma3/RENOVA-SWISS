# SITE.md — Renova Swiss

## 1. Vision
Site vitrine premium pour entreprise de rénovation en Suisse romande. Cible : propriétaires de biens immobiliers, projets de 20k–200k CHF. Ton : professionnel, premium, rassurant.

## 2. Stitch Project ID
> **À remplir** : créer un projet TEXT_TO_UI sur stitch.google.com, puis noter l'ID ici.
> Remplacer aussi dans `stitch.json`.

## 3. Tech Stack
- Single HTML file : `index.html`
- Assets locaux : `assets/` (photos) + `assets/gallery/` (galeries projets)
- Pas de framework JS — vanilla uniquement
